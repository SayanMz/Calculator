# Java Calculator (Android Native)
A clean, functional Android calculator built with **Java** and **XML**. This project was developed as part of my **BCA** coursework to demonstrate core Android development concepts, including UI layouts and event-driven programming.

## 🚀 Features
* **Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.
* **History Tracking**: Displays the previous calculation in a dedicated `TextView`.
* **Data Precision**: Utilizes `long` for integer operations and `double` for decimal calculations to ensure accuracy.
* **Responsive Design**: Uses a `GridLayout` / `TableLayout` approach for a consistent button interface.

## 🛠️ Tech Stack
* **Language**: Java
* **Platform**: Android Studio (Native)
* **UI**: XML Layouts
* **Architecture**: MVC (Model-View-Controller) pattern
